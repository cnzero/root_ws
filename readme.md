#  语音交互方案
1. 唤醒。
   
   当收听到唤醒词“你好小艺”时，对外发布：a-唤醒标志位，b-唤醒方位角。

2. 控制，对外发布如下字符串，代表的意义如下：
   - 字符串“L1”. “打开左灯”，“打开左边的灯”
   - 字符串“L0”. “关闭左灯”，“关闭左边的灯”
   - 字符串“R1”. “打开右灯”，“打开右边的灯”
   - 字符串“R0”. “关闭右灯”，“关闭右边的灯”
